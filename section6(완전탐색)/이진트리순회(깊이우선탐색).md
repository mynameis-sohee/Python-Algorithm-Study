# 이진트리순회
if else로 구현(초보자?)

### 이진트리
```python
      1
    /   \
   2     3
  / \   /  \
 4   5 6    7
```

## 전위 순회: 부모/왼쪽/오른쪽
1 2 4 5 3 6 7

```python
def DFS(x):
    if x > 7:
        return
    else:
        print(x, end=" ") #(본연의 작업) DFS의 방문: (전위순회)출력
        DFS(x*2) #왼쪽 노드
        DFS(x*2+1) #오른쪽 노드
if __name__ == "__main__":
    DFS(1)
```

## 중위 순회: 왼쪽/부모/오른쪽

## 후위 순회: 왼쪽/오른쪽/부모
