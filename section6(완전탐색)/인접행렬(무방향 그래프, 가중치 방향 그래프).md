# 인접 행렬
## 1. 무방향 그래프

  1---2 <br>
  | &nbsp; &nbsp;  | <br>
  3---4---5

||1|2|3|4|5|
|--|--|--|--|--|--|
|1| |1|1|||
|2|1|||1||
|3|1|||1||
|4||1|1||1|
|5||||1||

```python
n, m = map(int, input().split())
g = [[0] *(n+1) for _ in range(n+1)] #0으로 초기화된 2차원 리스트
for i in range(m):
  a, b = map(int, input().split())
  g[a][b] = 1
  g[b][b] = 1

for i in range(1, n+1):
  for j in range(1, n+1):
    print(g[i][j], end = " ")
  print()
```
## 2. 가중치 방향 그래프
||1|2|3|4|5|
|--|--|--|--|--|--|
|1| |1|1|||
|2|1|||1||
|3|1|||1||
|4||1|1||1|
|5||||1||
```python
#가중치 방향 그래프
n, m = map(int, input().split()) #m: 간선의 수
g = [[0] * (n+1) for _ in range(n+1)]
for i in range(m):
  a, b, c = map(int, input().split())
  g[a][b] = c

for i in range(1, n+1):
  for j in range(1, n+1):
    print(g[i][j], end = " ")
  print()
```
