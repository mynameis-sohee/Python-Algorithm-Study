## 결정 알고리즘(이진검색 중)이 사용되는 경우
답이 특정 범위 안에 있다는 것을 알 수 있음
중앙값을 정해놓고 이게 답이 되는지 안 되는지 확인해서 
범위를 반씩 줄여보면서 답을 찾음


### 내 풀이법
```python

k, n = map(int, input().split())

l_array = []
for _ in range(k):
    num = int(input())
    l_array.append(num)

#이진 탐색을 위한 시작점, 끝점
start = 0
end = max(l_array)

result = 0
while start <= end:
    total = 0 # 랜선 개수
    mid = (start + end) // 2
    for i in l_array:
        #잘랐을 때 랜선 개수
        if i > mid:
            total += (i // mid)

    if total < n:
        end = mid -1
    else: #total >= n
        result = mid
        start = mid + 1

print(result)
```
